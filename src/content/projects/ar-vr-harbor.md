## Overview

An innovative internal R&D project at ICT Group exploring how AR/VR technology could transform harbor management operations. The application creates an immersive digital twin of a harbor where managers can:

- View real-time vessel positions and berth occupancy in 3D
- Navigate the virtual harbor using VR headsets
- Use AR overlays on tablets for on-site inspection support
- Access operational data through interactive 3D dashboards

## Technical Implementation

Built with Unity 3D (C#) and Vuforia for AR marker tracking. Real-time data was pushed via WebSocket connections from the harbor management backend. The 3D environment was built with realistic harbor assets and interactive UI elements embedded in the 3D space.

This project was one of my first explorations into spatial computing and real-time 3D visualization — skills I later applied to other interactive data visualization projects.
